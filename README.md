# MediChain: Blockchain-Based Medical Prescription System

## Project Overview
This is a Final Year Project (FYP) utilizing blockchain technology to manage medical prescriptions securely.

## Tech Stack
- **Frontend:** React + Vite
- **Blockchain:** (e.g., Ethereum/Solidity)

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
